\# MangaSystemPlatform



Microservice-based manga creation workflow and publishing management system.



\## Overview



MangaSystemPlatform is a platform for managing the full manga creation and publishing workflow, including series planning, chapter production, page annotation, assistant task assignment, editorial review, publishing schedule, ranking, and AI-assisted manga segmentation.



\## Main Structure



\- `Client/`: Frontend application built with Next.js.

\- `Server/`: Backend microservices built with ASP.NET Core and FastAPI.

\- `docs/`: Project documentation.



\## Core Features



\- Manga series management

\- Chapter and page management

\- Manga page annotation

\- Task assignment for assistants

\- Editorial review workflow

\- Publishing and ranking management

\- AI-assisted manga region segmentation



\## Tech Stack



\- Frontend: Next.js, TypeScript, Tailwind CSS

\- Backend: ASP.NET Core, C#

\- Database: PostgreSQL

\- Gateway: YARP

\- Message Broker: RabbitMQ

\- Cache: Redis

\- File Storage: MinIO

\- AI Service: Python FastAPI
