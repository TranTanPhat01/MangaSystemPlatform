# Notification Service

Notification Service persists user notifications and consumes integration events from RabbitMQ.

## Projects

- `Manga.Notification.Api`
- `Manga.Notification.Application`
- `Manga.Notification.Domain`
- `Manga.Notification.Infrastructure`

## Local endpoints

- API: `http://localhost:5210`
- Swagger: `http://localhost:5210/swagger`

## Database

Connection string name: `NotificationDb`

Database: `NotificationDB`
