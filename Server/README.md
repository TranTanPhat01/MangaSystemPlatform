\# Server



Backend system for MangaSystemPlatform.



\## Architecture



The backend follows a simplified microservice architecture.



\## Main Components



\- API Gateway

\- Identity Service

\- Manga Management Service

\- Editorial Service

\- File Service

\- AI Service

\- Shared libraries

\- Infrastructure configuration

\- Backend tests



\## Tech Stack



\- ASP.NET Core

\- C#

\- PostgreSQL

\- Entity Framework Core

\- RabbitMQ

\- Redis

\- MinIO

\- SignalR

\- Docker

\- Python FastAPI for AI Service
